# 47_左侧当前区域四角角标补实现确认_0414

## 1. 本次补充说明

左侧“当前所在的二级区域”四个圆弧角括号，之前确实没有真正挂到选中项节点上。

这次已经补齐为真正可显示的实现，而不是仅停留在样式设想。

---

## 2. 实际修改位置

涉及文件：

- `Scripts/UI/GamePlayUI.Actions.cs`
- `Scripts/UI/GamePlayUI.Layout.cs`

### 2.1 新增公共绘制方法

在 `Scripts/UI/GamePlayUI.Actions.cs` 中新增：

- `CornerBracketPosition`
- `CreateCornerBracketOverlay(...)`
- `CreateCornerBracketPiece(...)`

作用：

- 分别生成左上、右上、左下、右下四个小范围橙色角标
- 每个角标只覆盖圆角附近的一小截边缘
- 保持“括号式角标”的感觉，而不是整圈描边

### 2.2 挂接到左侧二级区域选中项

在 `Scripts/UI/GamePlayUI.Layout.cs` 的 `BuildSecondaryAreaNode(...)` 中：

- 当 `isSelected == true` 时
- 额外执行 `panel.AddChild(CreateCornerBracketOverlay(new Color("#f6a53b"), 16, 2, 2));`

这样当前所在二级区域会同时显示：

- 原有深蓝选中底色
- 四角橙色定位角标

---

## 3. 当前视觉意图

这套做法不是给整块背景加完整橙色边框，而是只在四角补一小段高识别度提示。

好处是：

1. 保留原有深蓝选中底色，不破坏现有分层
2. 比整圈描边更克制，不会让左栏显得太重
3. 玩家能更快从左侧竖栏里定位“当前所在区域”
