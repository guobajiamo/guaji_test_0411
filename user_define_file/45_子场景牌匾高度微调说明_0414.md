# 45_子场景牌匾高度微调说明_0414

## 1. 本次调整原因

在上一轮完成：

- 牌匾式背景条
- 浮雕文字
- 文字垂直位置上移

之后，用户继续反馈：

- 文字依旧略微超出“牌匾”范围
- 当前牌匾本体本身还是有点太挤

因此这次不再继续硬挪文字，而是直接把牌匾本体略微增高几个像素。

---

## 2. 本次调整内容

调整文件：

- `Scripts/UI/GamePlayUI.Actions.cs`
- `Scripts/UI/GamePlayUI.Layout.cs`

具体改动：

1. 给牌匾样式增加少量上下内容边距  
   让牌匾背景条本身更有容纳空间。

2. 把标题承载层 `titleLayer` 的高度略微加高  
   让浮雕字有更自然的落位空间，不再显得被压扁。

---

## 3. 当前结果

当前小场景标题区域应当同时满足：

- 背景仍然是低矮牌匾，不会重新变成厚按钮
- 浮雕文字更容易被完整包进牌匾范围
- 视觉上不再那么挤

这次思路的关键是：

- 不继续只调文字
- 而是让牌匾本身稍微去适配文字

---

## 4. 验证结果

本次修正后已执行：

结果：

- 编译通过
- UI 冒烟测试通过

说明本次牌匾高度微调没有破坏当前 UI 主流程。
