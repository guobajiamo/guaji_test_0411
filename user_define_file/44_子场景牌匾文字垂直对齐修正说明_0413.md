# 44_子场景牌匾文字垂直对齐修正说明_0413

## 1. 本次修正问题

在上一轮把小场景标题做成牌匾浮雕字之后，出现了一个新的细节问题：

- 标题文字整体略偏下
- 浮雕字的部分像素超出了狭窄牌匾背景条的可视范围

这会让标题看起来像“压不进牌匾里”。

---

## 2. 本次修正内容

调整文件：

- `Scripts/UI/GamePlayUI.Actions.cs`
- `Scripts/UI/GamePlayUI.Layout.cs`

具体修正：

1. 将浮雕文字层整体上移  
   在创建标题文字层时，把纵向偏移统一往上收了一点。

2. 压低标题文字承载层高度  
   让承载标题文字的 `titleLayer` 更贴合窄牌匾本身的高度，不再给文字预留过高空间。

这样处理后，标题字会更自然地落在牌匾内部，而不是悬在偏下的位置。

---

## 3. 当前结果

当前小场景标题已经同时满足：

- 保持低矮牌匾结构
- 保持浮雕字效果
- 文字不会明显超出牌匾背景条

也就是说，现在的方向是：

- 牌匾条窄
- 文字略浮雕
- 文字垂直位置正确

---

结果：
