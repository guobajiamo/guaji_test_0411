# 55_左侧区域栏技能模式切换与技能聚合显示说明_0415
## 1. 本次调整目标
本次实现了 `user_define_file/54_左侧区域左边栏内部新增切换技能模式_功能需求.txt` 提出的核心需求：

1. 左侧栏支持 `区域 / 技能` 双模式切换。
2. 技能模式下按技能分组显示已习得技能。
3. 中间 `当前场景` 页签在技能模式下动态显示为 `<技能>相关`。
4. 技能模式下聚合显示该技能在各区域可见的相关子场景与事件。
5. 区域 NEW 提示在技能模式下迁移到左栏 `区域` 按钮旁闪烁显示。
6. 新增两个砍树测试采集点：`苹果树`、`瘴气树`。

## 2. 新增与改动的核心数据接口
### 2.1 UI 运行态字段（可存档）
文件：`Scripts/Core/Runtime/PlayerUiState.cs`

新增字段：

- `IsSkillSidebarMode`：左栏是否为技能模式。
- `SelectedSkillId`：技能模式下当前选中技能 ID。

### 2.2 存档读写同步
文件：`Scripts/Core/SaveLoad/SaveManager.cs`

已将以下字段纳入存档快照：

- `UiState.IsSkillSidebarMode`
- `UiState.SelectedSkillId`

读档时自动恢复上述字段；旧存档无该字段时走默认值（区域模式、空技能选择），不影响兼容。

## 3. 左侧栏 UI 与交互逻辑
文件：`Scripts/UI/GamePlayUI.cs`、`Scripts/UI/GamePlayUI.Layout.cs`

### 3.1 左栏顶部从单标题改为双按钮
- 左栏顶部改为两个切换按钮：`区域`、`技能`。
- 切换后自动刷新左栏树与中间场景页。
- 切换行为由 `SetLeftSidebarMode(bool isSkillMode)` 统一处理。

### 3.2 区域模式（保持原行为）
- 继续显示：一级区域 + 二级区域。
- 二级区域点击后进入该区域对应的当前场景视图。
- 二级区域上的 `New` 逻辑保持原实现。

### 3.3 技能模式（新行为）
- 左栏显示：技能分组（一级）+ 技能（二级）。
- 仅显示已习得技能（`PlayerSkillState.Level > 0`）。
- 分组与技能排序采用定义顺序（与习得先后无关）：
  - 先 `GroupOrder`
  - 再 `GroupName`
  - 再 `SkillOrder`
  - 再源文件顺序与技能 ID 兜底
- 分组支持展开/折叠状态。

## 4. 中间页签与内容联动逻辑
文件：`Scripts/UI/GamePlayUI.Actions.cs`、`Scripts/UI/GamePlayUI.Layout.cs`

### 4.1 页签标题动态化
- 当左栏为技能模式时：
  - `current_region` 页签标题改为 `<选中技能名>相关`。
  - 若当前无可选技能，则显示 `技能相关`。

### 4.2 技能模式下的场景聚合
- 技能模式进入 `RefreshSkillModeCurrentRegionPage()` 分支。
- 按 UI 布局定义顺序遍历所有可见区域/子场景。
- 筛选 `LinkedSkillId == 当前选中技能` 的可见事件（包含可互动与不可互动）。
- 以“子场景卡片”形式汇总显示，支持直接点击执行原事件逻辑。

## 5. NEW 提示迁移规则
文件：`Scripts/UI/GamePlayUI.Layout.cs`

### 5.1 区域模式
- `New` 继续显示在对应二级区域节点右侧（原行为）。

### 5.2 技能模式
- 由于区域节点隐藏，若任一区域仍有 `New`：
  - 在左栏 `区域` 按钮旁显示闪烁 `New`。
- 切回区域模式后，`New` 回到对应二级区域节点显示。
