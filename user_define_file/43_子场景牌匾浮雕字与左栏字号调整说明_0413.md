# 43_子场景牌匾浮雕字与左栏字号调整说明_0413

## 1. 本次调整目的

这一轮继续围绕“子场景标题更像牌匾”来做增强，同时顺手修正两个相关问题：

- 小场景标题的文字还不够像牌匾上的雕刻字
- 左侧区域栏二级区域文字偏小
- 中间“右键场景标题可以切换收藏置顶”这句提示会挤占视觉焦点

所以本次主要处理了 3 件事：

1. 给小场景标题文字增加浮雕感
2. 放大左侧二级区域字号
3. 去掉中间那句提示小字

---

## 2. 子场景标题文字浮雕化

调整文件：

- `Scripts/UI/GamePlayUI.Actions.cs`
- `Scripts/UI/GamePlayUI.Layout.cs`

### 2.1 做法

本次没有采用动画、闪烁或渐变，而是使用“分层文字”方式模拟牌匾浮雕：

- 底层：右下方向的暗色阴影字
- 中层：左上方向的浅色高光字
- 顶层：正常主文字

三层文字叠在一起后，会形成一种：

- 中间主字清晰
- 左上有受光
- 右下有压影

的效果，看上去会更像从牌匾上“凸出来”的字。

### 2.2 当前视觉方向

当前标题文字更接近：

- 轻浮雕字

而不是：

- 发光字
- 彩虹字
- 闪烁字

这样能保持牌匾感，又不会破坏整体 UI 的阅读稳定性。

---

## 3. 左侧二级区域字号放大

调整文件：

- `Configs/UI/gameplay_ui_theme.yaml`

本次把：

- `secondary_peace.font_size`
- `secondary_battle.font_size`

从之前较小的字号提高到了更明显的尺寸。

这样现在左侧二级区域名在它自己的背景行内，会更接近“占据大约三分之二行高”的感觉，读起来更醒目，也更符合当前界面已经变紧凑后的视觉比例。

---

## 4. 去掉中间提示小字

调整文件：

- `Scripts/UI/GamePlayUI.Layout.cs`

原本中间区域标题下方会显示：

- `右键场景标题可切换收藏置顶。`

这句提示虽然功能上有用，但它确实会：

- 占据本该给小场景标题的视觉注意力
- 拉开中间区域标题和子场景牌匾之间的空间

因此本次直接将其隐藏，不再显示。

这样处理后，中间区域顶部的注意力会更集中在：

- 当前区域标题
- 各个小场景牌匾标题

而不会被一行说明文字切走焦点。
